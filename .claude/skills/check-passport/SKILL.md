# /check-passport — Проверка паспортных данных

Полная проверка паспорта: реестр МВД, кем выдан, ИНН. Комбинирует 3 инструмента DaData MCP.

## Используемые инструменты

| Инструмент | Стоимость | Что делает |
|------------|:---------:|------------|
| `clean_passport` | 0.20 ₽ | Проверка по реестру недействительных МВД |
| `find_fms_unit` | Бесплатно | Кем выдан по коду подразделения |
| `find_inn_by_passport` | Бесплатно | ИНН по паспортным данным (через ФНС) |

## Алгоритм

1. Используй `clean_passport` для проверки серии и номера по реестру МВД
2. Если есть код подразделения — `find_fms_unit` для определения кем выдан
3. Если нужен ИНН — `find_inn_by_passport` (требуются ФИО + дата рождения + серия/номер)

## Интерпретация

| qc | Значение |
|:--:|----------|
| 0 | Паспорт действителен |
| 1 | Паспорт недействителен (в реестре МВД) |
| 10 | Не удалось проверить (реестр недоступен) |

## Формат ответа

```
## Результат проверки паспорта

**Серия и номер**: [XXXX XXXXXX]
**Статус**: [ДЕЙСТВИТЕЛЕН / НЕДЕЙСТВИТЕЛЕН / НЕ ПРОВЕРЕН]
**Выдан**: [название органа]
**Код подразделения**: [XXX-XXX]
**ИНН владельца**: [ИНН] (если запрошено)

### Вердикт
[Краткая оценка]
```

## Важно

- Данные паспортов — персональные данные (152-ФЗ)
- Используйте только с согласия владельца
- Результат проверки носит информационный характер

## Примеры

```
/check-passport 4510 235857
/check-passport 4510 235857 770-001
```
