# /validate-address — Валидация и стандартизация адреса

Полная проверка адреса: стандартизация, геокодирование, почта, логистика. Комбинирует до 8 инструментов DaData MCP.

## Используемые инструменты

| Инструмент | Стоимость | Что делает |
|------------|:---------:|------------|
| `suggest_address` | Бесплатно | Автодополнение адреса |
| `clean_address` | 0.20 ₽ | Стандартизация в 80+ полей |
| `find_by_id_address` | Бесплатно | Адрес по коду ФИАС/КЛАДР |
| `geolocate_address` | Бесплатно | Обратное геокодирование |
| `ip_locate` | Бесплатно | Город по IP-адресу |
| `find_postal_unit` | Бесплатно | Ближайшее почтовое отделение |
| `find_delivery_city` | Бесплатно | ID города в СДЭК/Boxberry/DPD |
| `suggest_country` | Бесплатно | Справочник стран |

## Алгоритм

1. Определи тип входных данных:
   - **Текст** → `suggest_address` → `clean_address`
   - **Координаты** → `geolocate_address`
   - **IP-адрес** → `ip_locate`
   - **Код ФИАС/КЛАДР/кадастровый** → `find_by_id_address`
2. Стандартизируй через `clean_address` для кодов качества
3. Если нужна почта — `find_postal_unit`
4. Если нужна доставка — `find_delivery_city`

## Коды качества

| qc | Значение |
|:--:|----------|
| 0 | Адрес распознан полностью (высокая достоверность) |
| 1 | Распознан с допущениями (средняя) |
| 2 | Не распознан (низкая) |

| qc_geo | Значение |
|:------:|----------|
| 0 | Точные координаты (дом) |
| 1 | Ближайший дом |
| 2 | Координаты улицы |
| 3-5 | Координаты города/региона |

## Формат ответа

```
## Результат валидации адреса

**Исходный**: [ввод пользователя]
**Стандартизированный**: [полный адрес]
**Достоверность**: [ВЫСОКАЯ / СРЕДНЯЯ / НИЗКАЯ]

**Компоненты**:
- Регион: [регион]
- Город: [город]
- Улица: [улица]
- Дом: [дом], Квартира: [кв]
- Индекс: [индекс]

**Геоданные**:
- Координаты: [lat, lon]
- Часовой пояс: [timezone]
- Ближайшее метро: [станция] ([расстояние])

**Логистика**:
- СДЭК: [id]
- Boxberry: [id]
- DPD: [id]

**Почтовое отделение**: [индекс, адрес]
```

## Примеры

```
/validate-address мск сухонская 11 кв 89
/validate-address 55.7558, 37.6173
/validate-address 46.226.227.20
```
