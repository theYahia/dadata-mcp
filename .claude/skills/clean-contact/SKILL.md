# /clean-contact — Проверка и очистка контактных данных

Валидация ФИО, телефона, email. Комбинирует до 6 инструментов DaData MCP.

## Используемые инструменты

| Инструмент | Стоимость | Что делает |
|------------|:---------:|------------|
| `clean_person` | 0.20 ₽ | Композитная проверка (в 5-8 раз дешевле) |
| `clean_name` | 0.20 ₽ | Разбор ФИО, пол, склонение |
| `clean_phone` | 0.20 ₽ | Оператор, регион, тип |
| `clean_email` | 0.20 ₽ | Исправление опечаток, тип |
| `suggest_fio` | Бесплатно | Автодополнение ФИО |
| `suggest_email` | Бесплатно | Автодополнение email |

## Алгоритм

1. Определи что дано:
   - **Всё вместе** (ФИО + телефон + email + адрес + паспорт) → `clean_person` — в 5-8 раз дешевле
   - **Только ФИО** → `clean_name` или `suggest_fio` (бесплатно)
   - **Только телефон** → `clean_phone`
   - **Только email** → `clean_email` или `suggest_email` (бесплатно)
   - **Частичное ФИО** → `suggest_fio` для дополнения

## Формат ответа

```
## Результат проверки контакта

### ФИО
- **Полное**: [Фамилия Имя Отчество]
- **Пол**: [мужской/женский]
- **Родительный падеж**: [для документов]

### Телефон
- **Номер**: [+7 (XXX) XXX-XX-XX]
- **Тип**: [мобильный/стационарный]
- **Оператор**: [название]
- **Регион**: [регион]

### Email
- **Адрес**: [email]
- **Тип**: [личный/корпоративный/одноразовый]
- **Статус**: [валидный/невалидный]

### Итого
- Проблемы: [список или "не обнаружены"]
```

## Примеры

```
/clean-contact Иванов Иван Иванович +79001234567 ivan@mail.ru
/clean-contact +79001234567
/clean-contact test@gmial.com
```
