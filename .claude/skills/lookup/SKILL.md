# /lookup — Справочники и статус аккаунта

Поиск по 12 российским справочникам и проверка аккаунта DaData. Комбинирует 3 инструмента DaData MCP.

## Используемые инструменты

| Инструмент | Стоимость | Что делает |
|------------|:---------:|------------|
| `lookup_reference` | Бесплатно | Поиск по 12 справочникам |
| `get_balance` | Бесплатно | Баланс и статистика за день |
| `get_versions` | Бесплатно | Даты обновления справочников |

## Справочники

| Запрос пользователя | directory | Пример |
|---------------------|-----------|--------|
| Вид деятельности | `okved2` | "разработка ПО", "62.01" |
| Продукция, услуги | `okpd2` | "программное обеспечение" |
| Муниципальное образование | `oktmo` | "Москва" |
| Станция метро | `metro` | "Арбатская" |
| Налоговая инспекция | `fns_unit` | "7700", "ИФНС 46" |
| Таможенный орган | `fts_unit` | "Шереметьевская" |
| Суд | `region_court` | "Арбитражный суд Москвы" |
| Валюта | `currency` | "доллар", "USD" |
| Товарный знак | `mktu` | "программное обеспечение" |
| Профессия | `profession` | "токарь", "сварщик" |
| Должность | `position` | "бухгалтер" |
| Мед. должность | `medical_position` | "терапевт" |

## Формат ответа

### Для справочников:
```
## Результат поиска: [справочник]

**Запрос**: [что искали]
**Найдено**: [количество]

1. **[код]** — [название]
2. **[код]** — [название]
```

### Для статуса аккаунта:
```
## Статус аккаунта DaData

**Баланс**: [сумма] ₽
**Запросов сегодня**: [кол-во] из [лимит]

**Обновления справочников**:
- ФИАС: [дата]
- ЕГРЮЛ: [дата]
- Банки: [дата]
```

## Примеры

```
/lookup разработка программного обеспечения
/lookup --balance
/lookup --versions
/lookup USD --directory currency
/lookup Арбатская --directory metro
```
