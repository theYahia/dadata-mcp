# /check-vehicle — Распознавание автомобиля

Определение марки и модели из текста. Комбинирует 2 инструмента DaData MCP.

## Используемые инструменты

| Инструмент | Стоимость | Что делает |
|------------|:---------:|------------|
| `clean_vehicle` | 0.20 ₽ | Распознавание марки и модели |
| `suggest_car_brand` | Бесплатно | Автодополнение марок |

## Алгоритм

1. Если полное описание — `clean_vehicle` для распознавания
2. Если частичный ввод или только марка — `suggest_car_brand` для подсказок

## Формат ответа

```
## Распознавание автомобиля

**Исходный ввод**: [что ввёл пользователь]
**Марка**: [brand]
**Модель**: [model]
**Стандартизированное**: [полное название]
**Статус**: [Распознан точно / С допущениями / Не распознан]
```

## Примеры

```
/check-vehicle тойота камри
/check-vehicle BMW X5
/check-vehicle бмв х5
```
