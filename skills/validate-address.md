# Валидация и стандартизация адреса

Полная проверка адреса: стандартизация, геокодирование, почтовое отделение, логистика.

## Инструкция

1. Получи от пользователя адрес в любом формате (строка, координаты или IP)
2. Определи тип входных данных:
   - **Текстовый адрес** → начни с `suggest_address`, затем `clean_address`
   - **Координаты** → используй `geolocate_address`
   - **IP-адрес** → используй `ip_locate`
   - **Код ФИАС/КЛАДР/кадастровый** → используй `find_by_id_address`
3. Если адрес найден через suggest — стандартизируй через `clean_address` для получения кодов качества
4. Если нужна почта — найди ближайшее отделение через `find_postal_unit`
5. Если нужна доставка — определи ID города в СДЭК/Boxberry/DPD через `find_delivery_city`
6. Если нужна страна — используй `suggest_country`

## Оценка качества

Интерпретируй коды качества из clean_address:
- **qc = 0** — адрес распознан полностью (высокая достоверность)
- **qc = 1** — адрес распознан с допущениями (средняя достоверность)
- **qc = 2** — адрес не распознан (низкая достоверность)
- **qc_geo = 0** — точные координаты (дом)
- **qc_geo = 1** — ближайший дом
- **qc_geo = 2** — координаты улицы
- **qc_geo = 3-5** — координаты города/региона

## Формат ответа

```
## Результат валидации адреса

**Исходный**: [что ввёл пользователь]
**Стандартизированный**: [полный адрес]
**Достоверность**: [ВЫСОКАЯ / СРЕДНЯЯ / НИЗКАЯ]

**Компоненты**:
- Регион: [регион]
- Город: [город]
- Улица: [улица]
- Дом: [дом], Квартира: [кв]
- Индекс: [индекс]

**Геоданные**:
- Координаты: [lat, lon]
- Часовой пояс: [timezone]
- Ближайшее метро: [станция] ([расстояние])

**Логистика** (если запрошено):
- СДЭК: [id]
- Boxberry: [id]
- DPD: [id]

**Почтовое отделение**: [индекс, адрес]
```
