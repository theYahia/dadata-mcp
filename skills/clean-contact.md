# Проверка и очистка контактных данных

Валидация ФИО, телефона, email одним запросом или по отдельности.

## Инструкция

1. Получи от пользователя контактные данные (любую комбинацию): ФИО, телефон, email, дата рождения, адрес, паспорт
2. Определи что дано и выбери стратегию:
   - **Всё вместе** (ФИО + телефон + email + адрес + паспорт) → используй `clean_person` — это в 5-8 раз дешевле раздельных запросов
   - **Только ФИО** → `clean_name` (платно) или `suggest_fio` (бесплатно, только автодополнение)
   - **Только телефон** → `clean_phone`
   - **Только email** → `clean_email` (платно) или `suggest_email` (бесплатно)
   - **Частичное ФИО** → `suggest_fio` для дополнения

## Анализ результатов

### ФИО (clean_name / suggest_fio)
- Разбивка: фамилия, имя, отчество
- Пол: мужской / женский / не определён
- Склонение по падежам (родительный, дательный, творительный)

### Телефон (clean_phone)
- Формат: +7 (XXX) XXX-XX-XX
- Тип: мобильный / стационарный
- Оператор и регион
- Часовой пояс

### Email (clean_email)
- Исправленные опечатки (если были)
- Тип: личный / корпоративный / одноразовый
- Качество: валидный / невалидный

## Формат ответа

```
## Результат проверки контакта

### ФИО
- **Полное**: [Фамилия Имя Отчество]
- **Пол**: [мужской/женский]
- **Родительный падеж**: [для документов]

### Телефон
- **Номер**: [+7 (XXX) XXX-XX-XX]
- **Тип**: [мобильный/стационарный]
- **Оператор**: [название]
- **Регион**: [регион]

### Email
- **Адрес**: [email]
- **Тип**: [личный/корпоративный/одноразовый]
- **Статус**: [валидный/невалидный]

### Итого
- Проблемы: [список или "не обнаружены"]
```
