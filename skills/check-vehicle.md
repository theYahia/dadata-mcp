# Распознавание автомобиля

Определение марки и модели автомобиля из текстовой строки.

## Инструкция

1. Получи от пользователя описание автомобиля в любом формате:
   - "тойота камри" → распознает марку и модель
   - "BMW X5" → распознает на английском
   - "бмв х5" → распознает транслит
   - Частичный ввод → используй `suggest_car_brand` для автодополнения
2. Используй `clean_vehicle` для полного распознавания марки и модели
3. Если пользователь ещё вводит или не уверен в марке — используй `suggest_car_brand` для подсказок

## Формат ответа

```
## Распознавание автомобиля

**Исходный ввод**: [что ввёл пользователь]
**Марка**: [brand]
**Модель**: [model]
**Стандартизированное**: [полное название]

### Статус
[Распознан точно / Распознан с допущениями / Не распознан]
```
