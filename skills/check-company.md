# Проверка контрагента

Полная проверка компании или ИП по ИНН/ОГРН с оценкой рисков.

## Инструкция

1. Получи от пользователя ИНН, ОГРН или название компании
2. Если дано название — используй `suggest_company` для поиска, предложи варианты
3. Используй `find_company_by_id` для получения полной информации по ИНН/ОГРН
4. Используй `find_affiliated` для поиска аффилированных компаний (если доступно)
5. Используй `find_brand` для получения бренда и логотипа (если нужно)
6. Если дан email — используй `find_company_by_email` для поиска компании
7. Если это ИП — проверь `find_self_employed` на статус самозанятого
8. Для белорусских компаний используй `suggest_company_by`, для казахстанских — `suggest_company_kz`
9. Если нужны банковские реквизиты — используй `find_bank`

## Анализ и вердикт

На основе полученных данных оцени:

- **Статус**: действующая / ликвидирована / в процессе ликвидации
- **Возраст**: когда зарегистрирована (менее 1 года — повышенный риск)
- **Руководитель**: есть ли данные о CEO, не является ли массовым
- **Адрес**: заполнен ли юридический адрес, не массовый ли
- **Финансы**: выручка, количество сотрудников (если доступно)
- **ОКВЭД**: основной вид деятельности
- **Аффилированность**: связанные компании, их статус

## Формат ответа

Выдай структурированный отчёт:

```
## Отчёт по контрагенту

**Компания**: [название]
**ИНН**: [инн] | **ОГРН**: [огрн]
**Статус**: [действующая/ликвидирована]
**Дата регистрации**: [дата] ([возраст])
**Руководитель**: [ФИО, должность]
**Юр. адрес**: [адрес]
**Основной ОКВЭД**: [код — описание]
**Сотрудники**: [кол-во]

### Оценка риска: [НИЗКИЙ / СРЕДНИЙ / ВЫСОКИЙ]

**Факторы**:
- [список факторов риска или надёжности]

### Аффилированные компании
- [список если есть]
```
