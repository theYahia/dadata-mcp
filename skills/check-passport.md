# Проверка паспортных данных

Полная проверка паспорта: реестр МВД, кем выдан, ИНН по паспорту.

## Инструкция

1. Получи от пользователя паспортные данные (серия и номер)
2. Используй `clean_passport` для проверки по реестру недействительных паспортов МВД
3. Если известен код подразделения — используй `find_fms_unit` для определения кем выдан паспорт
4. Если нужен ИНН владельца — используй `find_inn_by_passport` (потребуются ФИО, дата рождения, серия и номер паспорта)

## Интерпретация результатов

### Статус паспорта (clean_passport)
- **qc = 0** — паспорт действителен (не найден в реестре недействительных)
- **qc = 1** — паспорт недействителен (найден в реестре МВД)
- **qc = 10** — не удалось проверить (реестр недоступен)

### Код подразделения (find_fms_unit)
- Формат: XXX-XXX (например, 770-001)
- Возвращает полное название органа, выдавшего паспорт

### ИНН по паспорту (find_inn_by_passport)
- Работает через API ФНС — доступность не гарантирована
- Требует: фамилия, имя, отчество, дата рождения, серия, номер паспорта

## Формат ответа

```
## Результат проверки паспорта

**Серия и номер**: [XXXX XXXXXX]
**Статус**: [ДЕЙСТВИТЕЛЕН / НЕДЕЙСТВИТЕЛЕН / НЕ ПРОВЕРЕН]

**Выдан**: [название органа]
**Код подразделения**: [XXX-XXX]

**ИНН владельца**: [ИНН] (если запрошено)

### Вердикт
[Краткая оценка: паспорт валиден / есть проблемы]
```

## Важно

- Данные паспортов являются персональными данными (152-ФЗ)
- Используйте проверку только с согласия владельца паспорта
- Результат проверки по реестру МВД носит информационный характер
